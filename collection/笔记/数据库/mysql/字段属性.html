列属性又称之为字段属性，在mysql中一共有6个字段属性：null、默认值、列描述、主键、自动增长和唯一键

Null

如果对应的值为YES表示该字段可以为NULL，值为NO表示该字段不可以为NULL，

| num   | tinyint(4) unsigned zerofill | YES  |     | NULL    |       |
+-------+------------------------------+------+-----+---------+-------+
1 row in set (0.00 sec)

注意：

  1. 在设计表的时候，尽量不要让数据为空
  2. Mysql的记录长度为65535个字节，如果一个表中有字段允许为NULL，那么系统就会设计保留一个字节来存储NULL，最终有效存储长度为65534个字节。

    ->   name varchar(10) NOT NULL      -- 不能为空
    -> );

默认值

Default：默认值，在设计字段时，如果允许默认条件下用户不进行数据插入且用户确实没有插入数据，那么就使用事先准备好的数据来填充，通常填充的是NULL

使用示例：

1
2
3
4
mysql> CREATE TABLE my_default(
    ->   age int DEFAULT 18              -- 表示如果当前字段值在进行数据插入的时候没有提供，那么就使用18
    -> );
Query OK, 0 rows affected (0.04 sec)

复合主键

当一个字段无法确定唯一性的时候，需要其他字段来一起形成唯一性，就是说用来组成唯一性的字段如果有多个就是复合主键

案例：有一张学生选修课表，一个学生可以选修多个选修课，一个选修课也可以由多个学生来选，但是一个学生在一个选修课中只有一个成绩

复合主键的添加方法：在所有字段之后增加primary key选项“primary key(字段名1,字段名2,...)”

主键约束

  1. 当前字段对应的数据不能为空
  2. 在整张表中当前字段对应的数据不能重复

主键分类

主键分类采用的是主键所对应的字段的业务意义分类

  • 业务主键：主键所在的字段，具有业务意义（学生ID，课程ID）
  • 逻辑主键：自然增长的整型（应用广泛）

自动增长

自动增长：auto_increment，当给定某个字段该属性之后，该列的数据在没有提供确定数据时，系统会根据之前已经存在的数据进行自动增加后填充数据

通常自动增长用于逻辑主键

原理

  1. 在系统中有维护一组数据，用来保存当前使用了自动增长属性的字段，记住当前对应的数据值，再给定一个指定的步长
  2. 当用户进行数据插入的时候，如果没有给定值，系统在原始值上再加上步长变成新的数据
  3. 自动增长的触发：给定属性的字段没有提供值
  4. 自动增长只适用于数值

唯一键

唯一键：unique key，用来保证对应的字段中的数据在整张表中唯一的，虽然主键也可以用来保证字段数据唯一性，但是一张表只有一个主键，而唯一键在一张表中却可以有多个

注意：唯一键允许字段数据为NULL，且NULL可以有多个（因为NULL不参与比较）

创建唯一键

创建唯一键与创建主键非常类似

  1. 创建表时直接在表字段之后增加唯一键标识符：unique[ key]
    1

    mysql> CREATE TABLE my_unique1(
        ->   id int PRIMARY KEY AUTO_INCREMENT,
        ->   username varchar(10) UNIQUE
        -> );
    Query OK, 0 rows affected (0.07 sec)
  2. 创建表时在所有的字段之后使用unique key(字段列表)
    1

        ->   id int PRIMARY KEY AUTO_INCREMENT,

        -> );

    和主键一样允许复合唯一键

删除唯一键

一个表中允许存在多个唯一键：假设命令与主键一样：“ALTER TABLE 表名 DROP UNIQUE KEY;”，通过这条命令显然可以看出系统不可能找得到你想要删除哪个唯一键，所以这条命令是错误的

删除的基本语法：ALTER TABLE 表名 DROP INDEX 唯一键名;

Index关键字：索引，唯一键是索引的一种（索引可以提升查询效率）

复合唯一键

唯一键与主键一样可以使用多个字段来共同保证唯一性

一般主键都是单一字段（逻辑主键），而其他需要唯一性的内容都是由唯一键来处理

具体使用方法可以参考复合主键的使用方法
