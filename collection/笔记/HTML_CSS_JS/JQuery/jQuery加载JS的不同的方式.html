jQuery加载JS的不同的方式

2018-11-15
作者：loubth
为了不用把script标签写在body标签末尾而可以写在head标签中，我们可以将想要运行的代码绑定在载入事件处理函数里，这样就可以让js代码即使写在头部也可以最后运行 
//DOM中页面加载事件--------页面全部加载完毕才触发(标签,文字,图片,引入的文件)
window.onload = function () {
    console.log("小苏好帅");
};
//与上面相对应的jQuery中第一种页面加载的事件
$(window).load(function () {
    console.log("小苏好猥琐哦");
});

以上两种方式效果相同，都是在页面全部加载完毕才触发(标签,文字,图片,引入的文件)

jQuery中有一种更快的加载方法

//jQuery中第二种页面加载的事件----比上面的快---页面中的基本的元素加载后就触发
$(document).ready(function () {
    console.log("123456");
});

这种方式有两种简写形式

  1. 简写形式1 
    //这种方法的简写方式1
    jQuery(function () {
        console.log("页面加载了1111");
    });
  2. 简写形式2 
    //这种方法的简写方式2
    $(function () {
        console.log("页面加载了2222");
    });

我们一般采用简写形式2，因为其最简单
