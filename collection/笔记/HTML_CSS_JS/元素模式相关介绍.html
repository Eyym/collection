元素模式相关介绍

2018-08-10
作者：loubth

１.1 元素显示方式

 ☞ 块级元素
          ☞ 行内元素
          ☞ 行内块元素

１.2 块级元素

   代表：
              div, p,ul,ol,dl,li,dd,dt,h1...h6....
        特点：
                ☞ 块级元素独占一行显示
                ☞ 块级元素的默认宽度与其父元素的宽度一致
                ☞ 块级元素可以设置宽度和高度

 

１.3 行内元素

      代表：
                 span， a    , font, strong, i,s....
          特点：
                 ☞ 行内元素在一行上显示
                 ☞ 行内元素不能设置宽度和高度

１.4 行内块元素

    代表：
             img, input....
         特点：
              ☞ 元素可以在一行上显示
              ☞ 元素可以设置宽度和高度

              ☞ 元素不继承父元素的宽度和高度

１.5 元素模式转换

   ☞ 转化为块级元素（以块级元素的方式去显示）
              display: block;
           
         ☞ 转化为行内块元素
                 display: inline-block;
             
          ☞ 转化为行内元素
                display: inline;
       

             注意：
                  1. 可以把行内元素，块级元素，转化为任何一种元素
                  2. 行内块元素转化行内元素没有效果
                  3. 一般情况下会将行内元素进行转化
