public private protected和默认（友好型）的区别（同一个包下就是朋友）

Java中对类以及类中的成员变量和成员方法通过访问控制符（access specifier）进行区分控制。刚开始学Java语言的同学可能对public、privat、protected以及默认（一般称包访问权限）的区别不是很清楚，尤其是protected和默认的区别。本篇经验就来说说这个事儿。
  1. public是都能访问
  2. protected是只要本包内就能访问，包括本包内的子类和普通类，另外别的包内的子类也能访问，但是别的包的普通类不能访问
  3. 默认（包访问权限）是只有本包内才能访问，包括本包内的子类和普通类
  4. private是只有本类中才能访问（即使本包下的子类也不能访问）
