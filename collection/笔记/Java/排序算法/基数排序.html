基数排序

2019-07-01
loubth

介绍

基数排序(Radix Sort)是1887年赫尔曼·何乐礼发明的稳定的排序算法。它是传统桶排序的扩展，速度很快。基数排序是经典的空间换时间的方式，占用内存很大, 当对海量数据排序时，容易造成OutOfMemoryError

思路

基本思想

总思想：将所有待比较数值统一为同样的数位长度，数位较短的数前面补零。然后，从最低位开始比较，依次进行一次次排序。这样从最低位排序一直到最高位排序完成以后, 数列就变成一个有序序列

实现方式：首先设置十个一维数组充当编号为0~9的“桶”，在比较的过程中，将比较位放到与编号相等的桶里，然后以编号从小到大的顺序将桶中的数据放回原数组中，经过这样若干轮操作后，也就是最高位比较完成后，再次从桶中一次取出数据放回原数组，我们发现我们得到了一个有序序列

简单描述：从个位开始比较，把个位小的数放到前面，形成一个新的序列。然后比较新序列的十位，把十位小的数放在前面。依次类推，直到比到最高位为止，此时形成的序列，就是一个排序好的序列

示例（给序列 [123,54,8,4,27,54,99,666] 排序）

public static void radixSort(int[] arr) {
 
    //得到数组中最大的数的位数
    int max = arr[0]; //假设第一个数是最大的数
    for (int i = 1; i < arr.length; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    int maxLength = (max + "").length();    //maxLength就表示最大的数的位数
 
    //定义一个二维数组，表示10个桶，为了防止数据溢出，每个桶大小定为arr.length
    int[][] bucket = new int[10][arr.length];
 
    //定义一个一维数组来记录各个桶的中的数据个数。比如：bucketElementCounts[0]=3表示bucket[0]桶中有3个数据
    int[] bucketElementCounts = new int[10];
 
    //开始排序
    for (int i = 0, n = 1; i < maxLength; i++, n *= 10) {
        //(针对每个元素的对应位进行排序处理)， 第一次是个位，第二次是十位，第三次是百位...
        for (int j = 0; j < arr.length; j++) {
            //取出每个元素的对应位的值
            int digitOfElement = arr[j] / n % 10;
            //放入到对应的桶中
            bucket[digitOfElement][bucketElementCounts[digitOfElement]++] = arr[j];
        }
        //按照这个桶的顺序，取出数据放入原数组中
        int index = 0;
        //遍历每一个桶，并将桶中的数据，放入到原数组
        for (int k = 0; k < 10; k++) {
            //如果桶中，有数据，我们才放入到原数组
            if (bucketElementCounts[k] != 0) {
                //循环该桶即第k个桶(即第k个一维数组), 放入
                for (int l = 0; l < bucketElementCounts[k]; l++) {
                    //取出元素放入到arr
                    arr[index++] = bucket[k][l];
                }
            }
            //把桶清空
            bucketElementCounts[k] = 0;
        }
    }
}
