选择排序

2019-06-28
作者：loubth

思路

选择排序（select sorting）的基本思想是：选择最前面的数字充当临时最小值，然后遍历其余数字，如果比它更小，就将这个更小的数字记录下来当最小值，直到遍历完所有数字，此时就获得了这些数字中的真正的最小值，将当前记录的最小值的位置与序列最前面的数字的位置进行交换。然后对其余数字重复此过程，最终完成排序

过程：第一次从arr[0]~arr[n-1]中选取最小值，与arr[0]交换，第二次从arr[1]~arr[n-1]中选取最小值，与arr[1]交换，第三次从arr[2]~arr[n-1]中选取最小值，与arr[2]交换，…，第i次从arr[i-1]~arr[n-1]中选取最小值，与arr[i-1]交换，…, 第n-1次从arr[n-2]~arr[n-1]中选取最小值，与arr[n-2]交换，总共通过n-1次，得到一个按排序码从小到大排列的有序序列

示例（将下列一组数据“[4,8,7,3,2,9,6,5,1]”通过选择排序法整理成升序排序）
第1轮相邻元素交换，第1小的元素被选出放到前面   [1, 8, 7, 3, 2, 9, 6, 5, 4]
第2轮相邻元素交换，第2小的元素被选出放到前面   [1, 2, 7, 3, 8, 9, 6, 5, 4]

public static void selectSort(int[] arr) {
 
    //选择排序时间复杂度是O(n²)
    for (int i = 0; i < arr.length - 1; i++) {  //共有arr.length个数字需要排序，只需要选出前面arr.length-1个小的数字，最后一个自然是最大的
        //首先在未被选出的数字中选择最前面的数字充当最小值，并记录其下标
        int min = arr[i];
        int minIndex = i;
 
        //在未被选出的数字中寻找真正的最小值及其下标
        for (int j = i + 1; j < arr.length; j++) {  //arr[i]不需要与自己比较，下标从i+1开始即可
            if (arr[j] < min) { //说明假定的最小值，并不是最小
                min = arr[j]; //重置min
                minIndex = j; //重置minIndex
            }
        }
 
        //将选出的最小值放到未被选出的数字的最前面，也就是说与最前面的数字进行位置交换
        if (minIndex != i) {    //判断选出的最小值的位置是不是在最前面，是就不用交换了
            arr[minIndex] = arr[i];
            arr[i] = min;
        }
    }
 
 
}
