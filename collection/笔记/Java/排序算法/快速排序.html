快速排序

2019-06-30
作者：loubth

介绍

快速排序（Quick Sort）是对冒泡排序的一种改进。冒泡排序是交换相邻元素的位置，而快速排序是设置一个中点元素（枢纽），让比中点元素小的元素和比中点元素大的元素交换位置

思路

基本思想

首先要明确的问题：在非升序序列向升序序列的转变过程中，当一个值的左边的值都不大于这个值，右边的值都不小于这个值，那么这个值就处在了正确的位置（最终完成排序后枢纽这个值应该在的位置）

总思想：在一组无序的序列中选取一个值（通常是最左边的）作为“枢纽”，把小于这个枢纽的值放在这个枢纽的左边，大于这个枢纽的值放在这个枢纽的右边，这样操作一轮后，这个枢纽所处的位置就是“正确”的位置（最终完成排序后枢纽这个值应该在的位置）。然后给这个枢纽左右两边的数分别分成一组，进行同样的操作（选新枢纽，把小于新枢纽的放左边，大于的放右边），这样递归操作后，所有的值都将处在“正确”的位置上，整个序列的排序也就完成了

细节：如何把小于这个枢纽的值放在这个枢纽的左边，大于这个枢纽的值放在这个枢纽的右边呢？我们采取的做法是设置两个游标，一个从序列左边开始寻找比枢纽大的值，一个从序列右边开始寻找比枢纽小的值，当两者都找到目标值的时候，互换这两个值的位置。重复这个过程，直到游标相遇，此时游标相遇位置的左边的值都小于枢纽，右边的值都大于枢纽，因此两个游标所在的位置就是枢纽最终应该在的位置，我们就把枢纽放在这个位置上

public static void quickSort(int[] a) {
    quikeSort(a, 0, a.length - 1);
}
 
private static void quikeSort(int[] a, int low, int high) {
    //low和high代表数组的左边界和右边界的下标，low >= high 代表排序已经完成，不需要再排序了
    if (low >= high) {
        return;
    }
    //设置两个“指针”用来寻找需要的元素
    int left = low;
    int right = high;
 
    //保存枢纽值，这里选择数组最左边的元素充当枢纽
    int pivot = a[left];
 
    //将比枢纽值小的元素放在枢纽元素的左边，比枢纽值大的元素放在枢纽元素的右边
    while (left < right) {  //当left==right的时候，说明枢纽左边的值都小于枢纽值，枢纽右边的值都大于枢纽值，也就是说枢纽处在了正确的位置（最终排序完成的结果它应该在的位置）上
        //从后向前找到比枢纽小的元素
        while (left < right && a[right] >= pivot) {
            right--;
        }
        //把比枢纽小的元素放在左边（因为在上面的时候a[left]用来当枢纽pivot了，所以此处可以把a[left]当temp使用，方便下面交换）
        a[left] = a[right];
        //从前往后找到比枢纽大的元素
        while (left < right && a[left] <= pivot) {
            left++;
        }
        //vab比枢纽大的元素放在右边
        a[right] = a[left];
    }
    //放置枢纽值到它正确的位置上（因为此时left==right，所以a[left]或a[right]都是枢纽正确的位置）
    a[left] = pivot;
    //分治递归快排，直到整个序列有序
    quikeSort(a, low, left - 1);
    quikeSort(a, left + 1, high);
}
