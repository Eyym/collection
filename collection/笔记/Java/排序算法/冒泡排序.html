冒泡排序

2019-06-28
作者：loubth

思路

冒泡排序（Bubble Sorting）基本思想的是：从下标较小的元素开始，依次比较相邻元素的值，若发现逆序则交换，使值较大的元素逐渐从前移向后部，就象水底下的气泡一样逐渐向上冒

下面举一个极端的个例（把一组降序排列的数字通过冒泡排序改为升序），方便理解

示例（将下列一组数据“[9,8,7,6,5,4,3,2,1]”通过冒泡排序法整理成升序排序）

第1轮相邻元素交换，第1大的元素冒泡   [8, 7, 6, 5, 4, 3, 2, 1, 9]
第2轮相邻元素交换，第2大的元素冒泡   [7, 6, 5, 4, 3, 2, 1, 8, 9]
第3轮相邻元素交换，第3大的元素冒泡   [6, 5, 4, 3, 2, 1, 7, 8, 9]
第4轮相邻元素交换，第4大的元素冒泡   [5, 4, 3, 2, 1, 6, 7, 8, 9]
第5轮相邻元素交换，第5大的元素冒泡   [4, 3, 2, 1, 5, 6, 7, 8, 9]
第6轮相邻元素交换，第6大的元素冒泡   [3, 2, 1, 4, 5, 6, 7, 8, 9]
第7轮相邻元素交换，第7大的元素冒泡   [2, 1, 3, 4, 5, 6, 7, 8, 9]
第8轮相邻元素交换，第8大的元素冒泡   [1, 2, 3, 4, 5, 6, 7, 8, 9]

分析：共9个元素，所以最多需要8个元素冒泡即可完成排序，也就是说最多需要进行8轮相邻元素交换位置。第1轮相邻元素交换位置需要进行8次，随着冒了泡的元素的增多，每轮相邻元素交换位置的次数减少1次，直到第8轮只需要进行1次相邻元素交换位置就完成了全部元素的排序
所以说，对于一组有n个元素的序列，需要进行n-1轮相邻元素交换位置，第m轮元素交换需要进行n-m次相邻元素交换位置，如此操作后，无序序列就变成了有序数列

public static void bubbleSort(int[] arr) {
    //冒泡排序的时间复杂度 O(n²)
    int temp = 0;   //临时变量，用于交换
    for (int i = 0; i < arr.length - 1; i++) {  //正在进行第i+1轮交换，共有arr.length个元素，所以需要进行arr.length-1轮交换
        for (int j = 0; j < arr.length - 1 - i; j++) {  //正在进行第i+1轮交换的第j+1次交换，本轮共需要进行arr.length-1-i次交换
            if (arr[j] > arr[j + 1]) {  //这里的比较符号决定排序好的序列是升序还是降序
                //交换相邻元素
                temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

优化

因为排序的过程中，各元素不断接近自己的位置，如果一趟比较下来没有进行过交换，就说明序列有序，因此要在排序过程中设置一个标志flag判断元素是否进行过交换。从而减少不必要的比较。（这里说的优化，可以在冒泡排序写好后，再进行）

public static void bubbleSort(int[] arr) {
    //冒泡排序的时间复杂度 O(n²)
    int temp = 0;   //临时变量，用于交换
    boolean flag = false;   //标识变量，表示本轮排序是否进行过交换
    for (int i = 0; i < arr.length - 1; i++) {  //正在进行第i+1轮交换，共有arr.length个元素，所以需要进行arr.length-1轮交换
        for (int j = 0; j < arr.length - 1 - i; j++) {  //正在进行第i+1轮交换的第j+1次交换，本轮共需要进行arr.length-1-i次交换
            if (arr[j] > arr[j + 1]) {  //这里的比较符号决定排序好的序列是升序还是降序
                flag = true;    //标记本轮进行过交换，代表到目前为止序列还没有排序完成
                //交换相邻元素
                temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
 
        if (!flag) {    //判断本轮排序是否进行过交换，如果没有，证明全部排序已经完成
            break;  //在一趟排序中，一次交换都没有发生过
        } else {
            flag = false;   //重置flag，用来标记下一轮交换
        }
    }
}
